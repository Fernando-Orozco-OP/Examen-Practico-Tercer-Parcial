# Examen-Practico-Tercer-Parcial
Examen Practico de Programación en Web

# Gestor de Tareas

Aplicación web desarrollada con React + Vite que permite gestionar tareas de manera dinámica utilizando una API pública.

---

# Características

- React + Vite
- Hooks (`useState`, `useEffect`)
- Consumo de API con Axios
- Componentes reutilizables
- Formularios con validación
- Filtros dinámicos
- Diseño moderno y responsivo
- Renderización dinámica de listas

---

# Tecnologías Utilizadas

- React
- Vite
- Axios
- React Icons
- CSS3
